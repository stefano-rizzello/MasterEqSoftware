Description: Master Equations for River Networks

Files in this folder:

Mathematica/master_eq_river_network.nb
Mathematica/example.nb
A Mathematica library for computing dispersion and diffusion
of pollutants in river networks, with an example file.

Reduce/master_eq_river_network.red
Reduce/example.red
A Reduce library for computing dispersion and diffusion
of pollutants in river networks, with an example file.
See https://reduce-algebra.sourceforge.io/
for the Reduce Computer Algebra System.
